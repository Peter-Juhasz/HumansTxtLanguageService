# Humans.txt Language Servie
Visual Studio language service for humans.txt

Features:
 - Syntax highlighting, validation and code fixes
 - Brace matching
 - Outlining
 - Automatic formatting
 - Completion (well-known section titles)
 - Documentation, quick info
